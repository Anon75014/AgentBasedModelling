Model Structure

The model contains a grid of cells and a specified amount of farmers which own some cells.

The function calls used in a simulation are explained in Overview of CropWar.

Model

This file contains the Main structure of the Simulation. -> The AgentPy model

AgentPy model used for simulation.

An agent based model (ABM) to simulate competing farmers. :param parameters: dictionary of parameters, stored as self.p

setup()

Setting up of the ABM model.

step()

Move model from t to t+1.

Evolve the entire model by one time step: - Step all the “normal” farmers - Step potentially trained farmers to - Step the market - let farmers decide if they want to change crops - refresh the river water content

update()

Record the properties of the farmers each step.
