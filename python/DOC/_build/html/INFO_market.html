Market Description

The market.py file contains market class.

It’s functions get called by the CropwarModel and Farmer in order to update the market each time step with the current amount in stock -> supply.

Market

class market.Market(crop_sortiment: crops.CropSortiment, agents: agentpy.sequences.AgentList, base_demand: float, demand_fraction: float)

Market which regulates the prices of the crops

_calc_current_demand() → None

Calculates the current demand using an expansive market model

_calc_global_stock() → None

Calculates all the available resources

calc_global_price() → Dict[int, float]

Calculates the global price according to supply and demand

calc_global_supply() → Dict[int, float]

Calculates the total supply that is provided by the agents
